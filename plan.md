# 한국 사극 영화들을 시대별로 정리해서 시각화해주는 웹사이트를 개발

## 레이아웃
- 맨 위에는 제목 "사극영화 시간대별 정리"가 있고 그 아래 작은 글씨로 만든이(나)의 이름과 깃허브 하이퍼링크가 있다. 
- 가운데에는 가로로 긴 선이 타임라인 역할을 한다. 100년 단위로 시대가 구분되어있고, 간단하게 주요 사건들이 표시되어 있다. 
- 타임라인 위에는 각 사극 영화들의 포스터가 있고, 각 영화 속 사극의 시대가 어디부터 어디까지인지 표시되어 있다. 
- 예를 들어 <관상>이라는 영화는 계유정난과 관련된 영화이기 때문에 계유정난이 일어난 연도에 표시되어 있다. 
- 양쪽 화살표가 있어서 화살표를 누르면 다음 영화로 이동한다. 오른쪽 화살표를 누르면 지금 영화보다 바로 이후 시간대의 영화, 왼쪽은 반대이다. 
- 영화의 포스터를 클릭하면 해당 영화의 정보를 볼 수 있는 위키피디아나 왓챠피디아로 새 창을 띄워 이동한다. 

## 데이터 수집 방법
- 한국영화가 본격적으로 시작된 1980년부터 시작한다. 
- 1980년부터 1년씩 공개되었던/상영한 영화들 리스트를 만든다. 
- 리스트의 영화들의 왓챠피디아 링크 혹은 위키피디아 링크를 찾는다. 
- 리스트의 영화를 하나하나 검색하고 조사해서 어떤 시간대의 역사적 사건을 영화로 만들었는지 조사한다.
